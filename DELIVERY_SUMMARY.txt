VERIFICATION STEPS
================================================================================

1. TEST LAMBDA FUNCTIONS
   aws lambda invoke \
     --function-name ConsoleSensei-Scan \
     --payload '{"userId":"test-user"}' \
     response.json

2. TEST API GATEWAY
   curl -X POST https://YOUR_API_ID.execute-api.us-east-1.amazonaws.com/v1/scan \
     -H "Content-Type: application/json" \
     -d '{"userId":"test-user"}'

3. TEST COGNITO
   aws cognito-idp admin-create-user \
     --user-pool-id YOUR_USER_POOL_ID \
     --username testuser@example.com \
     --temporary-password TempPassword123!

4. RUN APPLICATION
   npm run dev
   # Open http://localhost:5173

================================================================================
TROUBLESHOOTING
================================================================================

ISSUE: AWS CLI not found
   SOLUTION: Install from https://aws.amazon.com/cli/

ISSUE: CDK bootstrap fails
   SOLUTION: Check credentials with: aws sts get-caller-identity
             Try again: cdk bootstrap aws://ACCOUNT_ID/REGION

ISSUE: Lambda deployment fails
   SOLUTION: Check build: ls -la backend-lambda/dist/
             Review logs: aws logs tail /aws/lambda/consolesensei --follow

ISSUE: API returns 403
   SOLUTION: Verify User Pool ID in .env.local
             Check JWT token validity

See DEPLOYMENT_CHECKLIST.md for detailed troubleshooting.
